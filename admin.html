📋 Guide de configuration
Option 1 — Brevo (gratuit jusqu'à 300 emails/jour)
1. Créez un compte sur brevo.com
2. Allez dans "Automatisation" → "Créer un workflow"
3. Déclencheur : "Webhook entrant" (connectez votre Stripe)
4. Action : "Envoyer un email" → collez le template ci-dessus
5. Dans Stripe → Webhooks → Ajouter endpoint → URL Brevo

Option 2 — Stripe + Email direct (le plus simple)
1. Dans Stripe → "Paramètres" → "Emails clients" → Activer
2. Stripe envoie automatiquement la confirmation de paiement
3. Vous, vous envoyez manuellement le code (vérifiez vos paiements chaque jour)

Option 3 — Zapier (automatisation sans code)
1. zapier.com → Nouveau Zap
2. Déclencheur : "Stripe — Paiement réussi"
3. Action : "Gmail — Envoyer un email" avec le template
4. Variable [CODE] : générez via ce dashboard → copiez dans le Zap
